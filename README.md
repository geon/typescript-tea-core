A library that implements a version of the Model View Update pattern, similar to what is used in The Elm Architecture.

## How to develop

Node version >=12.6.0 is needed for development.

To execute the tests run `yarn test`.

## How to publish

```
yarn version --patch
yarn version --minor
yarn version --major
```
